Бизнес-аналитик

Процессы · Excel · SQL · ТЗ
▾
📋

Одно из самых широких направлений. Такой специалист не только анализирует цифры, но и смотрит на бизнес-процессы в целом, работая со смежными командами над их улучшением. Python здесь обычно не нужен 🙂

Примеры задач
  • Считать экономическое обоснование проектов и оценивать их эффективность
  • Вести базы знаний команды в Confluence
  • Собирать требования от заказчиков и составлять ТЗ смежным командам
  • Моделировать бизнес-процессы «as is» и «to be» в нотациях
  • Анализировать и визуализировать данные, делать отчёты
Для этого нужно
  • Хорошо знать Excel: ВПР, условные формулы (ЕСЛИ, СЧЁТЕСЛИ, СУММЕСЛИ), сводные таблицы
  • Уметь визуализировать данные и презентовать их в PowerPoint — строить понятные слайды, графики, таблицы
  • Знать основы SQL: стандартные SELECT-запросы, фильтрация (WHERE), джойны (JOIN), группировка (GROUP BY)
  • Уметь описывать и структурировать бизнес-требования: писать ТЗ, user stories, acceptance criteria
Будет плюсом
  • Знакомство с инструментами моделирования бизнес-процессов: Business Studio, MS Visio, Aris, нотации BPMN и EPC
  • Знать принципы работы Tableau или PowerBI для визуализации и анализа процессов
  • Участие в кейс-чемпионатах, олимпиадах или научных конференциях — покажет умение решать нестандартные задачи
  • Опыт работы в Confluence, Jira или аналогах для ведения документации

Продуктовый аналитик

A/B-тесты · Python · Метрики продукта
▾
🚀

Помогает продуктовым командам принимать решения на основе данных: исследует поведение пользователей, строит воронки и проверяет гипотезы через A/B-тесты.

Примеры задач
  • Строить воронки конверсии и выявлять узкие места в продукте
  • Проектировать и анализировать A/B-эксперименты
  • Разрабатывать продуктовые метрики (DAU, Retention, LTV)
  • Готовить дашборды для продуктовых команд
  • Проводить когортный и сегментационный анализ пользователей
Для этого нужно
  • Уверенно знать SQL: оконные функции (ROW_NUMBER, LAG, LEAD), агрегации, подзапросы, работа с большими таблицами
  • Знать базовый Python: библиотеки Pandas для работы с данными, NumPy для вычислений
  • Понимать основы статистики: t-тест, p-value, доверительные интервалы, размер выборки для A/B-тестов
  • Знать продуктовые метрики: DAU/MAU, Retention, Churn, LTV, ARPU, конверсия по воронке
  • Уметь строить понятные презентации и дашборды для нетехнической аудитории
Будет плюсом
  • Опыт работы с Tableau или PowerBI: построение интерактивных дашбордов и фильтров
  • Продвинутые библиотеки Python: Matplotlib и Seaborn для визуализации, Scipy для статистики
  • Понимание продуктового мышления: фреймворки AARRR, HEART, Jobs To Be Done
  • Участие в кейс-чемпионатах или хакатонах по анализу данных

Дата-аналитик

Big Data · SQL · Python · Визуализация
▾
📊

Работает с большими объёмами данных: строит отчёты, находит закономерности и помогает бизнесу принимать решения, основанные на фактах, а не на интуиции.

Примеры задач
  • Прогнозировать показатели компании с помощью статистических моделей
  • Проводить разведочный анализ данных (EDA)
  • Строить и поддерживать аналитические дашборды
  • Анализировать большие объёмы данных в Vertica / ClickHouse
  • Автоматизировать рутинные отчёты с помощью Python
Для этого нужно
  • Продвинуто знать SQL: оконные функции, CTE (WITH-запросы), работа с Vertica и ClickHouse, оптимизация запросов
  • Знать Python на уровне уверенного использования: Pandas, NumPy, автоматизация отчётов, работа с API
  • Уметь визуализировать данные: Matplotlib, Seaborn, а также строить дашборды в Tableau или PowerBI
  • Знать Excel на продвинутом уровне, включая OLAP-кубы и сложные формулы массивов
  • Понимать основы статистики: описательная статистика, корреляция, регрессия, проверка гипотез
Будет плюсом
  • Знакомство с основами машинного обучения: линейная регрессия, классификация, кластеризация (scikit-learn)
  • Опыт работы с облачными платформами: Yandex Cloud, BigQuery, Spark
  • Участие в соревнованиях на Kaggle или хакатонах по Data Science
  • Знание основ Git для версионирования кода и совместной работы

Дата-инженер

ETL · Пайплайны · SQL · Python
▾
⚙️

Строит инфраструктуру данных: пайплайны, хранилища, ETL-процессы. Без дата-инженера аналитики не смогут получить нужные данные в нужном виде и в нужное время.

Примеры задач
  • Разрабатывать и поддерживать ETL/ELT-пайплайны
  • Проектировать схемы хранилища данных (DWH)
  • Обеспечивать качество и консистентность данных
  • Оптимизировать производительность SQL-запросов
  • Автоматизировать процессы загрузки и трансформации данных
Для этого нужно
  • Продвинуто знать SQL: сложные запросы, оптимизация, работа с партиционированием, индексами в ClickHouse и Vertica
  • Знать Python на продвинутом уровне: написание ETL-скриптов, работа с pandas, sqlalchemy, requests
  • Понимать архитектуру хранилищ данных: понятия DWH, Data Lake, Data Mart, слои staging/ods/dds/dm
  • Знать основы Linux: навигация, bash-скрипты, работа с cron для планирования задач
  • Уметь работать с Git: ветки, pull request'ы, code review
Будет плюсом
  • Опыт с Apache Airflow: создание DAG'ов, операторы, мониторинг пайплайнов
  • Знакомство с Apache Spark или Kafka для потоковой обработки данных
  • Базовые знания Docker и контейнеризации для деплоя пайплайнов
  • Понимание принципов dbt (data build tool) для трансформации данных

▾
📈

Создаёт точные финансовые модели и планы, на которые опирается топ-менеджмент при принятии стратегических решений.

Примеры задач
  • Планировать и прогнозировать финансовые показатели компании
  • Готовить и дорабатывать модели бюджетного планирования
  • Анализировать отклонения фактических значений от бюджетных (план-факт)
  • Разрабатывать сценарии развития бизнеса для топ-менеджмента
  • Придумывать методы оптимизации процессов планирования
Для этого нужно
  • Отлично знать Excel: не только сводные таблицы и условные функции, но и OLAP-кубы, функции ИНДЕКС/ПОИСКПОЗ, финансовые функции (NPV, IRR, PMT)
  • Уметь строить финансовые модели: P&L, Balance Sheet, Cash Flow, DCF-оценка
  • Хорошо знать SQL: агрегатные и оконные функции, работа с Vertica и ClickHouse для выгрузки данных
  • Иметь базовые знания экономики и финансов: понятия маржи, EBITDA, рентабельности, оборотного капитала
  • Уметь визуализировать данные и строить убедительные презентации в PowerPoint для топ-менеджмента
Будет плюсом
  • Python: библиотеки Pandas и NumPy для автоматизации расчётов, Matplotlib и Seaborn для визуализации
  • Знакомство с системами 1С или Axapta (Microsoft Dynamics) для работы с учётными данными
  • Участие в кейс-чемпионатах по финансам и инвестициям (CFA Institute, РБК и др.)
  • Базовое понимание МСФО (IFRS) или РСБУ для работы с финансовой отчётностью
